Projet Principal

Plateforme E-transport - Gestion de Coopérative

Application web complète pour la gestion d'une coopérative de transport avec système de réservation, gestion de véhicules et de clients

Page d'Accueil - Dashboard
Vue.js + Node.js + MySQL
Interface moderne avec vue d'ensemble du parc automobile, des clients, des réservations et des recettes
Description du projet

Développement d'une plateforme e-transport complète avec un système de gestion intégré pour une coopérative de transport. L'application permet de centraliser toutes les opérations dans une seule interface web moderne et intuitive :

  • Gestion complète du parc automobile
  • Gestion des clients et contacts
  • Système de réservation avancé
  • Suivi des paiements et génération de reçus
  • Statistiques et tableaux de bord
  • Génération de PDF (reçus)
  • Interface responsive et moderne
  • Base de données relationnelle
Technologies utilisées
Vue.js (Frontend) Node.js + Express (Backend) MySQL (Base de données) HTML5/CSS3/JavaScript Bootstrap 5 PDF Generation
